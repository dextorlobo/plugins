Upload your logo for WordPress login page instead of the usual WordPress logo with simple settings.

== Description ==

For those aiming to personalize their WordPress site, the CustomDash Plugin is indispensable. This plugin allows you to effortlessly replace your WordPress login page logo with a customized image or text logo in just a few clicks, without the need for coding expertise. Ensure your login page reflects your brand's uniqueness effortlessly.

This plugin offers exceptional user-friendliness, enabling you to swiftly upload your logo via the WordPress customizer. Moreover, it provides full customization capabilities, allowing you to fine-tune the logo's size, placement, and various other settings to seamlessly align with your site's design.

Beyond its user-friendly interface, the CustomDash Plugin is lightweight and quick to load, safeguarding your site's performance from any slowdowns. Moreover, it seamlessly integrates with all the newest versions of WordPress, ensuring uninterrupted access to the latest features and security enhancements.
	
== Installation ==

1. Login to your WordPress admin panel and go to Plugins -> Add New
2. Type **Custom Dash** in the search box and click on search button.
3. Find Change Custom Dash plugin.
4. Then click on Install Now after that activate the plugin.

== Frequently Asked Questions ==

= How to change Image Logo in CustomDash Plugin? =

1. Click "Settings" tab from left navigation menu:
2. Click "Custom Dash" menu under the Settings menu
3. Click "Image" radio button
4. Upload new logo in "Image Logo" using WP media uploader.
5. Click "Save Changes" button to save the configuration details.

= How to change Text Logo in CustomDash Plugin? =

1. Click "Settings" tab from left navigation menu:
2. Click "Custom Dash" menu under the Settings menu
3. Click "Text" radio button
4. Enter the text in "Text Logo" field.
5. Click "Save Changes" button to save the configuration details.
